Field Null Description Note

array of objects		 Y An array of UserSecurityQuestion objects  
status
integer		  

The status

  • 1 = Active
  • 2 = Deactive
  • 3 = PasswordNotSet
  • 4 = LoginAttemptFailed
  • 5 = AnswerAttempFailed
