ReportSavingParameter¶

Field NULL Description Note
saveAs
boolean		   Is this saved as new report  
report
object		   A ReportDefinition object  
section
integer		  

The report section

  • 0 = All
  • 1 = DataSource
  • 2 = Fields
  • 3 = CalculatedField
  
tenantId
string (GUID)		 Y The id of the tenant  
ignoreCheckChange
boolean		   Is the check for report changes ignored  
ignoreCheckName
boolean		   Is the check for report name ignored  
insertWithAssignedId
boolean		   Whether to use the populated report id in report object to insert  
overrideCalculatedFields
boolean		   Whether to override calculated fields  
reportState
integer		  
  • 0 = Posted - Report is posted from client
  • 1 = Saved - Get report from database
  • 2 = Draft - Get report from draft
  
isSavingReport
boolean		   Whether report is saving or not

Only validate report data source when saving report (isSavingReport = true)

New in version 2.6.8.
