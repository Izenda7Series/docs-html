Report¶

Field NULL Description Note
name
string		   The name of the report  
reportDataSource
array of objects		   An array of ReportDataSource objects  
type
integer		  

The report type

  • 0 = Reports
  • 1 = Templates
  
previewRecord
integer		   The number of records to preview  
advancedMode
boolean		   Is the report in advanced mode  
allowNulls
boolean		   Does the report display null values in left join  
isDistinct
boolean		   Does the report allow duplicated data  
categoryId
string (GUID)		 Y The id of the report category  
categoryName
string		   The name of the report category  
subCategoryId
string (GUID)		 Y The id of the report sub-category  
subCategoryName
string		   The name of the report sub-category  
tenantId
string (GUID)		 Y The id of the tenant if available  
tenantName
string		   The name of the tenant if available  
description
string		   The report description  
title
string		   The report title  
lastViewed
datetime		 Y The report’s last view date and time  
owner
string		   The name of report’s owner  
ownerId
string (GUID)		 Y The id of report’s owner  
headerContent
string		   The content of the header in json format  
footerContent
string		   The content of the footer in json format  
excludedRelationships
string		   The deleted relationships, which should not be displayed again  
numberOfView
integer		   The number of view  
renderingTime
real number		   The report rendering time  
createdById
string (GUID)		 Y Id of user who created this report  
modifiedById
string (GUID)		 Y Id of user who last modified this report  
snapToGrid
boolean		   Snap to Grid option  
usingFields
string		   The ids of the fields used in report, separated by comma
.. deprecated:: 2.0.0		  
hasDeletedObjects
boolean		   True if using objects in deleted connections  
header
object		   Header content in a dynamic object  
footer
object		   Footer content in a dynamic object  
titleDescription
object		   Title description content in a dynamic object  
sourceId
string (GUID)		 Y The id of the source report in case this was copied from that report  
checked
boolean		   Is selected for copy in Copy Management  
copyDashboard
boolean		   Whether this is copied because dashboard was copied  
exportFormatSetting
object		   An ExportFormatSetting object  
deletable
boolean		   True if report can be deleted  
editable
boolean		   True if report can be edited  
movable
boolean		   True if report can be moved  
copyable
boolean		   True if report can be copied  
accessPriority
integer		   The access priority  
active
boolean		   True if this is the latest version  
isGlobal
boolean		   Whether this is a global report  
isCheck
boolean		   True if this is selected in copy workspace

Field NULL Description Note
id
string (GUID)		   The id of this object
Example: 572bd576-8c92-4901-ab2a-b16e38144813		 Allow null incase insert a new entity
state
integer		  

The entity state of this object

  • 0 = None
  • 1 = Insert
  • 2 = Delete
  • 3 = Update
  
deleted
boolean		   Is this object deleted  
inserted
boolean		   Is this object inserted  
version
string		 Y The version  
created
datetime in ISO 8601 format		 Y The created datetime  
createdBy
string		   The creator  
modified
datetime in ISO 8601 format		 Y The modification datetime  
modifiedBy
string		   The user who last modified this object
