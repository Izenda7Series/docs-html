System Configuration/Async ExportingÂ¶

The System Configuration/Email page allows user to set up email sending options.

  1. In browser, log in to Izenda as a user with System Configuration permission.

  2. Click Settings, then System Configuration then Email in the left menu.

  3. Select the Setting Level: either System or a specific tenant.

    Custom Async Configuration

    For tenant level, select to re-use configuration from system or to define a separate one.

  4. Turn ON the Exporting Service and enter the details into the configuration seciton

  5. Save the configuration.
