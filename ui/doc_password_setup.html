Password Setup¶

Password Setup¶

The Password Setup page allows user to create password for the first time. The page can only be opened from a password link, which is a short-expiry web address uniquely created from the user information.

  1. Fig. 163 Password Setup Page

    Click the password link to open Password Setup page in browser.

  2. Verify the link by entering the exact information used to create the link and click Next.

  3. In the next screen, enter the password and answers to security questions if available.

    The password must meet the Password Complexity setting in Security Policies.

  4. Click OK to save.

Password Change for Current User¶

  1. Log in as the user having the password to change.

  2. Click My Profile under the user name at the top right of the screen.

  3. Click Security Options in the left menu, then click Change Password.

    Fig. 164 Security Options

  4. Fig. 165 Change Password Pop-up

    Enter the current password, then the new password twice to verify in the Change Password pop-up.

Password Reset for Another User¶

  1. In browser, log in to Izenda as a user with User Setup permission.
  2. Click Settings, then User Setup in the left menu.
  3. Click Configure Password Option.
  4. Click Generate Password Link.
  5. Click the copy icon (that looks a bit like this ❐) to copy the password link to clipboard.
  6. Paste and open the password link in browser to reset the password (similarly to Password Setup steps).
