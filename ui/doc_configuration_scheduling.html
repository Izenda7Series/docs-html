System Configuration/Scheduling¶

The System Configuration/Scheduling page allows user to:

  • search for schedule instances
  • view, edit and delete schedule instances

Search and View Schedule Instances¶

  1. In browser, log in to Izenda as a user with System Configuration permission.
  2. Click Settings, then System Configuration then Scheduling in the left menu.
  3. Select the Setting Level: either System or a specific tenant.
  4. The Search Criteria box is displayed on top.
  5. Select the Reporting Type: either Report, Dashboard or All from the drop-down, then enter a partial report/dashboard name to search for.
  6. Optionally select other criteria, then click the Search button.
  7. The matching schedules will be displayed in Search Result box.

Fig. 127 Schedule Instances Search

Edit Schedule Instance¶

  1. Click the pencil icon in Action column to edit a schedule.
  2. The Edit Schedule pop-up is similar to Report Designer.
  3. Click the OK button to save the schedule.

Delete Schedule Instances¶

  • Click the x icon in Action column to delete a schedule.
  • To delete multiple schedules:
    1. Tick the check-boxes in front of each one.
    2. Click the now enabled Delete button at the top of the list.
