Report List¶

The Report List page allows user to

  • browse reports by type, categories and sub-categories
  • view report history
  • copy and move reports between categories
  • print and export reports
  • set up subscriptions
  • delete reports
  • rename and delete report categories

Browse Reports¶

  1. In browser, log in to Izenda as a user with Report permission.

  2. Click Reports.

  3. The reports are listed in their respective categories and sub-categories.

  4. Select REPORTS or TEMPLATES to view reports of this type.

    Fig. 324 Report List

  5. Click the plus icon on the right to view report properties.

    Fig. 325 Report Properties in Report List

  6. The buttons at the bottom allow user to Open (in Report Viewer), Design, Copy or Move to another name or category, Print or Export to another format, add Subscriptions, or Detele the report.

View Report History¶

  1. Click the history icon (similar to ↺) next to version number to open Report History pop-up.

  2. The report versions together with modified date and user are displayed with the most current one on top.

  3. There are shortcuts for the current version to be opened in Report Designer, copied and moved, or deleted.

    Fig. 326 Report History

  4. The previous versions can be removed if no longer needed, or copied to another name or category then used in addition to the current version.

    Fig. 327 Report History - Copy Version

Search for Reports¶

The Search box at the top allows user to search for specific reports.

  1. Fig. 328 Report List -Search

    Select a specific element to search for in the dropdown on the left of the Search box. Default is All.

  2. Type a partial name and click the search icon (🔍).

  3. The matching reports will be displayed.

Rename Report Categories¶

  1. Fig. 329 Report List - Rename Category

    Click the pencil icon to rename a report category.

  2. Fig. 330 Report List - Confirmation New Category Name

    Enter the name in place and click the check icon to save.

  3. Fig. 331 Report List - Duplicated New Category Name

    The new name will be checked for duplication with existing categories.

Copy a Report¶

Fig. 332 Report Copy Pop-up

The Copy function allows user to copy a report to a new name and/or a new category and sub-category. leaving the original report intact.

  1. Click the Copy button to open Copy Report pop-up.
  2. Enter a new name for the report if changing the name.
  3. Enter or select a new category and sub-category if needed.
  4. The report name in will be checked for duplication in the destination category.
  5. Click OK to close the pop-up and perform the copy.

Move a Report¶

Fig. 333 Report Move Pop-up

The Move function allows user to move a report to a new category and sub-category, optionally changing the name in the process.

  1. Click the Move button to open Move Report pop-up.
  2. Enter a new name for the report if changing the name.
  3. Enter or select a new category and sub-category.
  4. The report name in will be checked for duplication in the destination category.
  5. Click OK to close the pop-up and perform the move.

Add Report Subscriptions for Current User¶

  1. Click the Subscribe button to open Report Subscription pop-up.

  2. The report’s current subscriptions are displayed with a summary of the schedule.

  3. Fig. 334 Report Subscription Pop-up

    Click Add Subscription button to open Add Subscription pop-up.

  4. Enter a name for this subscription.

  5. In Schedule tab:

    1. Select Subscription Type:
      • Subscription Alert: deliver to current user only if it has data at scheduled time.
      • Subscription Report: deliver to current user regardless whether it has data or not at scheduled time.
    2. Select a more suitable time zone than the default if needed.
    3. Select a start date in the future.
    4. Enter the start time (some common values can be quickly selected from the pre-defined list).
    5. Select a common recurrence period from the list, or select Custom Recurrence to define a different one.

    Fig. 335 Report Subscription Recurrence Sample

  6. In Delivery tab, select:

    • “Email” Delivery Type: send the subscription to the email address of current user.
      1. Select Delivery Method: email a Link, email an Attachment, or email with Embedded HTML body.
      2. Customize the default template if necessary.
    • “File Location” Delivery Type: save the subscription as a file.
      1. Delivery Method is Send to disk.
      2. Select the Export File Type: PDF, Word Doc, Excel or CSV.

    Fig. 336 Report Subscription Delivery

  7. Click OK to close Add Subscription pop-up.

  8. Click Save to close Report Subscription pop-up.

Remove Report Subscriptions for Current User¶

  1. Click the Subscribe button to open Report Subscription pop-up.
  2. The report’s current subscriptions are displayed with a summary of the schedule.
  3. Click the Delete icon (x) to delete each subscription.
  4. Click Save to close Report Subscription pop-up.

To delete multiple subscriptions:

  1. Tick the checkboxes beside each one.
  2. Click the now enabled Delete icon at the bottom (that looks like a waste basket)
  3. Click Save to close Report Subscription pop-up.

Fig. 337 Report Subscription Delete Multiple
