Role Setup¬∂

The Role Setup page allows user to manage roles.

List and View Roles¬∂

  1. In browser, log in to Izenda as a user with Role Setup permissions.
  2. Click Settings, then Role Setup in the left menu.
  3. Select the Setting Level: either System or a specific tenant.
  4. List of roles and statuses will be displayed in the Middle Panel.
  5. Click any role in the list.
  6. The role’s information will be displayed in the 4 tabs Data Model Access, Permissions, Users, and Permission Summary.

Fig. 112 Role Setup List and Details

Search for Roles¬∂

The Search box at the top allows searching for specific roles.

  1. Fig. 113 Search for Roles

    Type a partial name and click the search icon (üîç).

  2. Only matching roles will be displayed in the list.

  3. Clear the search box to see the full list.

Add Role¬∂

  1. Click the + Add Role button at the top.

  2. Enter a unique name for the role.

  3. Fig. 114 Data Model Access

    In Data Model Access tab, tick items in Available Data Sources then click the > icon to move to Visible Data Sources.

    It is possible to move only some fields of a table to Visible list. In this case, users of the role will only see visible fields in the table.

  4. In Permissions tab, configure permissions for this role similarly to Tenant Permissions.

  5. Fig. 115 Users tab

    In Users tab, tick users in Available Users then click the > icon to move to Assigned Users.

    Assigning users is not available when Izenda is integrated inside an existing system.

  6. The Permission Summary tab is a read-only comparison of permissions among all roles of the current tenant.

    Fig. 116 Permission Summary tab

    For example: a quick scan through this page ensures that mod002 role consists of User and Role setup permissions only while mod001 role consists of Data Model, Report and Dashboard permissions only.

  7. Click the Save button at the top.

  8. The new role will appear in the list in Middle Panel.

  9. Click the Activate link next to the role in Middle Panel to make it active.

Edit Role¬∂

  1. Click the role in the list in Middle Panel.
  2. The role’s information is displayed in the 4 tabs.
  3. Edit the information.
  4. Click the Save button at the top, then OK in the confirmation pop-up.

Deactivate Role¬∂

  1. Untick the Active checkbox (‚òë) of the role in Middle Panel.
  2. Click OK in the confirmation pop-up.
  3. The role becomes inactive. Its users no longer have the permissions specified in this role.

Note

The Cancel button at the top will have no effect in this case.

Delete Role¬∂

  1. Click the Delete icon (x) of each role.
  2. Click OK in the confirmation pop-up.
  3. The role is deleted and removed from the list in Middle Panel.

Note

The Cancel button at the top will have no effect in this case.
