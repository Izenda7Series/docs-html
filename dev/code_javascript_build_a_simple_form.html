Build a simple Form¶

Steps:

  1. Follow Build a Generic Report until the step to add report parts.
  2. Prepare an empty ReportPartForm object with default properties.
  3. For each selected data source fields, populate a ReportPartElement object with default properties.
  4. Add the ReportPartElement objects into columns.elements in ReportPartForm object.
  5. Update the properties of the map in properties field per user selection (See ReportPartFormProperties).
  6. Back to the steps in Build a Generic Report.

Populate selected data sources fields¶

  1. Refer to the similar step in Building a Grid guide to:

    1. Get the list of available data sources fields from POST report/availableQuerySourceFields
    2. Build a corresponding ReportPartElement object for each selected data source field
    3. Populate a default ReportPartElementProperties for properties field in each ReportPartElement object

    See Sample Properties for a ReportPartElement for some samples.

  2. Add the ReportPartElement objects into columns.elements in ReportPartForm object.

  3. Build the HTML to display selected data source fields in htmlContent.

    Note

    The double quotes must be escaped before adding to json, e.g. “htmlContent”: “<div><field class=\”field-wrapper\” ...”

    <div>
    <custom-HTML-tags...>
    <field class="field-wrapper" field-name="[connection].[schema].[table].[field 1]">
        <field-prop key="subTotalFieldDataType" value="Izenda data type for sub total calculation"></field-prop>
        <field-prop key="grandTotalFieldDataType" value="Izenda data type for grand total calculation"></field-prop>
        <field-prop key="fieldId" value="id of field 1"></field-prop>
        <field-prop key="dataFieldType" value="Izenda data type"></field-prop>
        <field-prop key="querySourceId" value="id of parent query source of field 1"></field-prop>
        Display text in Report Designer for field 1
    </field>
    </custom-HTML-tags...>
</div>

    The whole <field>..</field> tag will be replaced by the value of selected data source field in Report Viewer.
