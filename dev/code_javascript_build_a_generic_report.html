Build a Generic Report¶

  1. Save as draft the report with name, category and selected data sources
    1. Prepare an empty ReportDefinition object.
    2. Populate report name and categories.
    3. Populate selected data sources.
    4. Prepare a ReportSavingParameter object as API input.
    5. Call Save report as draft API, then receive back the ReportSavingResult object.
  2. Populate the draft with report parts
    1. Extract the report field from the ReportSavingResult object. This is a ReportDefinition object which has been assigned a report id.
    2. Populate the report parts:
      • Build a simple Grid
      • Build a simple Chart
      • Build a simple Gauge
      • Build a simple Map
      • Build a simple Form
  3. Save the report
    1. Prepare a ReportSavingParameter object as API input.
    2. Call Save report API.
