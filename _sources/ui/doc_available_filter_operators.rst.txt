Available Filter Operators
---------------------------------

Filter operators are used to designate a subset of data for analysis and modeling.

The following is a list of available filter operators for reference. Additional operators may be added in future versions of Izenda.

==========
Comparison
==========

* Name : Blank
	* Label : =
	* Allow Parameters : FALSE

* Name : Not Blank
	* Label : =
	* Allow Parameters : FALSE

* Name : Is Less Than
	* Label : <
 * Allow Parameters : FALSE

* Name : Is Greater Than
	* Label : >
	* Allow Parameters : FALSE

* Name : Between
	* Label : Between
	* Allow Parameters : FALSE

* Name : Not Less Than
	* Label : >=
	* Allow Parameters : FALSE

* Name : Not Greater Than
	* Label : <=
	* Allow Parameters : FALSE

* Name : Not Between
	* Label : Not Between
	* Allow Parameters : FALSE
